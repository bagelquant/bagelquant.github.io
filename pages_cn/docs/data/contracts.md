# Contracts

合同使数据集可重复。

## 数据集身份

按名称、提供程序、版本和可选快照标识逻辑数据集。

## 数据集架构

独立于物理存储声明字段和主键。

## 数据合约

结合身份、模式、所有者、新鲜度和元数据。

## 谱系记录

记录来源、操作、参数和创建时间。
