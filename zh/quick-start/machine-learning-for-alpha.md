机器学习可以将许多特征结合到回报预测中，但它并没有消除定量研究的困难部分：时间点数据、泄漏预防、验证设计、可解释性和成本意识评估。

从强大的线性基线开始，然后仅在研究问题证明合理时添加更灵活的模型。参见 [Machine Learning Models](/zh/learn/models/machine-learning/)。
